📲 Configurar Señales Futuras → Telegram

TradingView puede enviar las señales directamente a tu Telegram sin ningún servidor. Seguí estos pasos:
1. En TradingView, en el indicador T3-CCI → agregá este código al final del script para habilitar alertas:
buySignal = crossover(xccir, 0)
sellSignal = crossunder(xccir, 0)
alertcondition(buySignal, title="T3-CCI COMPRA", message='{"chat_id":"TU_CHAT_ID","text":"🟢 COMPRA BTC\\nPrecio: {{close}}\\nHora: {{timenow}}","parse_mode":"HTML"}')
alertcondition(sellSignal, title="T3-CCI VENTA", message='{"chat_id":"TU_CHAT_ID","text":"🔴 VENTA BTC\\nPrecio: {{close}}\\nHora: {{timenow}}","parse_mode":"HTML"}') 2. Creá la alerta en TradingView → Alerts → Create Alert → elegí la condición → activá Webhook URL
3. En el campo Webhook URL pegá:
https://api.telegram.org/botTU_BOT_TOKEN/sendMessage 4. ¡Listo! Cada señal llega directo a tu Telegram sin servidores.
